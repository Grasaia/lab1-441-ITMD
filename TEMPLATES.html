Resume Customization Templates

Use these templates as quick reference for common customizations.

Add Another Job Entry

Copy this block and paste it in the experience-section, after existing job items:

<article class="experience-item"> <h3 class="job-title">Your Job Title Here</h3> <p class="company">Company Name</p> <p class="dates">January 2024 - December 2024</p> <ul class="job-description"> <li>Key accomplishment or responsibility</li> <li>Another achievement or task</li> <li>Technology or process you used</li> </ul> </article>

Add Another Project

Copy this block and paste it in the projects-section, after existing projects:

<article class="project-item"> <h3 class="project-title">Project Name</h3> <p class="project-description"> Brief description of what the project does and what problem it solves. Keep this concise - 1-2 sentences. </p> <p class="project-tech"><strong>Technologies:</strong> HTML, CSS, JavaScript</p> <a href="https://github.com/yourusername/project-name" class="project-link" target="_blank">View on GitHub</a> </article>

Add Another Skill

Add this line to the skills-list ul element:

<li class="skill-item">Your Skill Name</li>

Example skills to add:

React.js Vue.js TypeScript Python SQL MongoDB AWS Docker REST APIs UI/UX Design Project Management Communication

Add a New Section (Certifications, Languages, etc.)

Add this entire section to your main element, before the footer:

<section class="certifications-section"> <h2 class="section-title">Certifications</h2> <article class="certification-item"> <h3 class="cert-title">Certification Name</h3> <p class="cert-issuer">Issuing Organization</p> <p class="cert-date">Completed: Month Year</p> </article> </section>

Then add CSS styling in styles.css:

.certifications-section { padding: 2rem; border-top: 1px solid #eee; } .certification-item { margin-bottom: 1.5rem; padding: 1rem; background-color: #f8f9fa; border-radius: 4px; }

Change Color Scheme

Find and replace these color codes in styles.css:

/* Original Blues */ #0066cc → Your Primary Color #0052a3 → Your Dark Color #61dafb → Your Accent Color /* Example: Change to Purple Theme */ #0066cc → #7c3aed (Purple) #0052a3 → #6d28d9 (Dark Purple) #61dafb → #a78bfa (Light Purple)

Change Fonts

In styles.css, modify the body font-family:

/* Current */ font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif; /* Modern Serif */ font-family: 'Georgia', 'Times New Roman', serif; /* Clean Sans-Serif */ font-family: 'Trebuchet MS', 'Arial', sans-serif; /* Google Fonts (add link in <head>) */ <link href="https://fonts.googleapis.com/css2?family=Poppins:wght@400;600;700&display=swap" rel="stylesheet"> font-family: 'Poppins', sans-serif;

Add More Contact Links

Add these lines to the contact-links nav:

<!-- Phone Number --> <a href="tel:+14155552671" class="contact-link"> <span>Phone</span> </a> <!-- Twitter/X --> <a href="https://twitter.com/yourusername" class="contact-link" target="_blank"> <span>Twitter</span> </a> <!-- Personal Website --> <a href="https://yourwebsite.com" class="contact-link" target="_blank"> <span>Website</span> </a>

Add Multiple Education Entries

<article class="education-item"> <h3 class="degree">Bachelor of Science in Information Technology</h3> <p class="institution">Illinois Institute of Technology (IIT)</p> <p class="graduation">Expected Graduation: May 2026</p> </article> <article class="education-item"> <h3 class="degree">High School Diploma</h3> <p class="institution">Lincoln High School</p> <p class="graduation">Graduated: June 2022</p> </article>

Create Custom Section Styling

For a new section with different styling:

/* HTML */ <section class="awards-section"> <h2 class="section-title">Awards & Recognition</h2> <article class="award-item"> <h3>Award Name</h3> <p>Awarded by: Organization</p> <p>Date: Month Year</p> </article> </section> /* CSS */ .awards-section { padding: 2rem; border-top: 1px solid #eee; } .award-item { background: linear-gradient(135deg, #fff3cd 0%, #fff9e6 100%); padding: 1rem; border-radius: 4px; margin-bottom: 1rem; }

Add Project Screenshots or Images

<article class="project-item"> <img src="project-screenshot.png" alt="Screenshot of project homepage showing header and navigation" class="project-image"> <h3 class="project-title">Project Name</h3> <p class="project-description">Description...</p> </article> /* Add to CSS */ .project-image { width: 100%; height: auto; border-radius: 4px; margin-bottom: 1rem; }
